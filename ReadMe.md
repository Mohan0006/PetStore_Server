
# ***INITIALIZE AND RUN CODE (LOCAL MACHINE)***

1. Open the backend folder in the VS Code

-> Install all dependencies

2. npm i

-> To run tests on the server

3. npm test

4. Ctrl + c to terminate the tests

-> To run the server 

5. npm start 

server will start at localhost:4000
